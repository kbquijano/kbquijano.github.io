A) HTML Structure - top

B) CSS Files and Structure - top

We use LESS preprocessor for all theme styles. Code is separated to three LESS files in "assets/less" folder - app.less, mixins.less and customselect.less. Main file is app.less, where you can find all variables, font imports and styles. In other two LESS files we saved mixins and custom styles for custom select plugin.

CSS files are generated to "assets/css" folder. There you can find all plugins CSS files as well. We generate LESS files with Koala app for Mac.

Primary theme color is orange. If you would like to change primary color you can define new one in app.less as @maincolor: #ff0000;
